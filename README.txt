Welcome to a little Python big data experimentation!

To run:
1) Create conda environment pointing to anaconda
2) Change scripts/config.py file to point your database
3) Download https://www.kaggle.com/currie32/crimes-in-chicago/downloads/crimes-in-chicago.zip, extract the 4 .csv files inside to a new folder called /data/chicago on the project root
4) Run dbinit.py
5) Run analysis.py
